# How to use the script!!!!



To run the installation process on your MatchX M2 Pro Gateway just use the following command
```
wget https://raw.githubusercontent.com/WantClue/MatchX/master/install.sh
```
```
chmod +x install.sh
```
```
./install.sh
```
